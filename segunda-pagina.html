O que são e como usar tags semanticas

As tags semanticas são nada mais nada menos do que um modo de facilitar a leitura do nosso codigo por outros desenvolvedores, tambem facilita com que os leitores de tela tenham uma facilidade maior ao ler nosso codigo para alguem com algum tipo de deficiencia visual, alem do mais isso nos ajuda a identificar mais facilmente os erros no codigos

Exemplos de tags semanticas

Na imagem podemos ver algumas tags muito usadas como:

  • 1. 
     <header> - Se encontra dentro de uma <section>, sua tradução literal é "cabeçario"
  e dentro desta tag podemos encontras as tags de <h1> até <h6> usado para colocar um titulo
  e a tag <p> usado para criar um paragrafo.

  • 2. 
     <nav> - É usado para criar um conjunto de links,
  é necessario se atentar que nem todos os links de um documunto ou site devem estar dentro da nav , dentro da <nav> podemos encontrar: <ul> e <li>
  usados para criar uma lista não ordenada e definir uma lista de itens respectivamente, tambem pode ser encontrada a tag <a href=""> 
  que é usada para colocar os links dentro das tags <ul> e <li>
